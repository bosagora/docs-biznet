    컨센서스 엔진

    다음과 같은 목표를 가지고 BizNet의 컨센서스 엔진을 설계하였습니다.

    1. 블록의 최종적으로 확정하기 위해 필요한 대기 시간을 줄입니다.
    2. 블록생성 주기는 이더리움 네트워크와 동일하기 합니다.
    3. 인플레이션은 없고, 블록 생성의 보상은 거래수수료입니다.
    4. 이더리움과 호환이 가능합니다.
